# 九宫格 小游戏

#### 介绍
用一个小游戏来验证Java模块化，非模块化，JavaFx，NetBeans IDE（11版本），Ant，Maven，Gradle的集成使用。

#### 软件架构
情况见各具体项目。目前还没有gradle项目。
